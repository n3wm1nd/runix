### DND wiki
keep track of game world information of a dnd campaign
#### transcript [llm task]
fetch the transcript from a youtube video, convert it into a usable format. filter out unrelevant information, fix transcription errors if it was autogenerated. start marking relevant characters, items and locations
#### summarize [llm task, read files]
use the world information, and the transcript, to summarize what happened that session, in this state, interlink the information from the transcript with existing wiki entries (or ones that should be created). put all that into a session entry
#### update wiki [llm task, dynamically read and update files]
update the wiki with the events of this session, fix connections and crossreferences, canonize names
update the summary page so next run will have enough context to orient itself in the world and interpret events correctly

### deep analysis of creator
go through the works of a creator end evaluate it based on criteria
for example: use transcripts of a youtube analyst to evaluate their trustworthiness and biases
#### summarization
go through each work, summarize it in a few sentences (with focus on the evaluation critera, but keeping enough context)
#### evaluation
use all summaries to evaluate the sum of this work, it's trend over time
